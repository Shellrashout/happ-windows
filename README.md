# Общие сведения

**Happ** — кроссплатформенный клиент для создания, хранения и управления пользовательскими профилями подключения, использующими конфигурации на базе технологии **Xray**.

Приложение предназначено для использования на мобильных устройствах и персональных компьютерах и предоставляет единый интерфейс для работы с параметрами соединения. Пользователь получает возможность самостоятельно создавать профили, импортировать готовые конфигурации, изменять настройки, сохранять различные наборы параметров и выбирать подходящий вариант для конкретного сценария.

Архитектура Happ построена на принципе независимых профилей. Каждый профиль содержит собственную конфигурационную структуру, включающую параметры подключения, правила маршрутизации, настройки обработки сетевых потоков и дополнительные элементы управления. Отдельное хранение профилей позволяет удобно организовывать несколько вариантов настроек, изменять их независимо и быстро переключаться между ними при необходимости.

# Функции управления профилями

Happ предоставляет пользователю инструменты для полноценной работы с конфигурациями:

* создание персональных профилей подключения;
* импорт настроек из локальных файлов;
* редактирование сохранённых конфигураций;
* хранение нескольких независимых наборов параметров;
* настройка правил маршрутизации;
* изменение дополнительных параметров соединения;
* выбор активного профиля;
* обновление существующих конфигураций;
* выполнение всех операций непосредственно на устройстве.

Каждая конфигурация сохраняется отдельно, что позволяет формировать удобную систему профилей для различных задач и условий использования.

# Хранение пользовательских данных

Все созданные профили и импортированные конфигурационные файлы сохраняются исключительно в локальном хранилище устройства.

Happ не использует внешние облачные платформы для хранения пользовательских настроек, не выполняет автоматический обмен конфигурациями между устройствами и не передаёт сведения о профилях сторонним сервисам.

Пользователь самостоятельно управляет всеми операциями с данными. Добавление, импорт, изменение, сохранение, удаление и активация профилей выполняются непосредственно внутри приложения.

Приложение не выполняет анализ содержимого импортируемых конфигураций, не использует содержащиеся в них данные для дополнительных целей и не предоставляет доступ к пользовательским настройкам другим лицам.

# Особенности работы

Happ является локальным инструментом для управления параметрами подключения и не предоставляет собственную сетевую инфраструктуру.

Приложение не содержит:

* собственных VPN-серверов;
* прокси-систем;
* удалённых узлов обработки или перенаправления данных;
* встроенных коллекций готовых конфигураций;
* механизмов автоматического получения профилей.

Все необходимые параметры пользователь добавляет самостоятельно — посредством ручного ввода либо загрузки заранее подготовленных конфигурационных файлов.

Happ не определяет назначение создаваемых профилей и не контролирует способы их использования. Пользователь самостоятельно отвечает за применение приложения с учётом законодательства и требований, действующих в его регионе.
